# PLC Exercises — Tom Mejer Antonsen

Solutions to exercises from **"Collection of Exercises for PLC Programming"** by Tom Mejer Antonsen (2024).

## About

This repository contains PLC programs written as part of working through the exercise book. The exercises cover logical circuit programming from beginner to advanced level, using real-world industrial scenarios.

## Platforms

| Platform | Software | Status |
|----------|----------|--------|
| Allen-Bradley | Studio 5000 (Logix Designer) | ✅ In progress |
| Siemens | TIA Portal | 🔲 Planned |

## Folder Structure

Each exercise has its own folder containing:

```
02.XX-Exercise-Name/
├── Studio5000/        # Allen-Bradley project files (.ACD, .L5X)
├── TIAPortal/         # Siemens project files (planned)
├── screenshots/       # Program screenshots
└── README.md          # Exercise description and I/O table
```
