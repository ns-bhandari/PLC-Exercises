# 02.04 — Turn on Light with One of the Two Push Buttons

**Book:** Collection of Exercises for PLC Programming — Tom Mejer Antonsen (2024)

---

## Exercise Description

Two push buttons **S1** and **S2**, and a lamp **P1** are connected to a PLC. Both push buttons are **Normally Open (NO)** contacts.

Write a PLC program where the lamp **P1** is turned on when **either** push button S1 **or** push button S2 is pressed.

---

## Inputs & Outputs

| Tag | Type | Description |
|-----|------|-------------|
| S1  | DI (NO) | Push button 1 |
| S2  | DI (NO) | Push button 2 |
| P1  | DO | Lamp |
