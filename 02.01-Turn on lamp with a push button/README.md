# 02.01 — Turn on Lamp with a Push Button

**Book:** Collection of Exercises for PLC Programming — Tom Mejer Antonsen (2024)

---

## Exercise Description

A push button switch **S1** and a lamp **P1** are connected to a PLC. The push button is a **Normally Open (NO)** contact.

Write a PLC program that meets the following requirements:
- The lamp **P1** is turned on when **S1** is pressed
- The lamp **P1** should be off when **S1** is not pressed

---

## Inputs & Outputs

| Tag | Type | Description |
|-----|------|-------------|
| S1  | DI (NO) | Push button switch |
| P1  | DO | Lamp |
